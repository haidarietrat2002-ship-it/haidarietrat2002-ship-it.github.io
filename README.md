# haidarietrat2002-ship-it.github.io

Overview

This project was developed as part of the Web Programming & Mobile Computing (WMC) module.
The goal of the project is to install, configure, and customize the flat-file CMS Grav in order to create a professional platform for documenting IT learning progress, programming knowledge, and training projects.
Additionally, this project challenges me to learn new technologies, improve my technical skills, and deepen my understanding of content management systems.

Grav was chosen because it is lightweight, fast, file-based (no database required), and highly flexible through themes and plugins
